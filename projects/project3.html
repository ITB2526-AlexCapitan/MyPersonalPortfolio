Personal Portfolio Website

Designed and developed a responsive professional portfolio to showcase technical projects. Built with clean HTML, CSS3, and JavaScript, focusing on performance and minimalism.

HTML CSS JavaScript Git

The Philosophy: "Vanilla" First

In an era dominated by heavy frameworks like React or Angular, I chose a different path for this project. The goal was to achieve the highest possible performance with zero "bloat".

By using Vanilla JavaScript and pure CSS, the site loads instantly, respects the user's battery life, and provides a fluid 60fps experience without the overhead of a Virtual DOM.

Visual Identity System

The aesthetic was heavily inspired by the Apple Design Guidelines: heavy use of negative space, san-serif typography, and subtle glassmorphism effects.

Technical Challenges

While the layout is minimal, the interactivity is complex. The centerpiece is the 3D MacBook integration using Three.js.

Scroll-Driven Animation: Unlike a simple video, the 3D model reacts to the user's scroll position. I mapped the scroll Y-offset to the rotation of the laptop lid mesh, creating a seamless "closing" animation as the user navigates down to the content.

Responsive design was handled using CSS Grid and Flexbox, ensuring the layout adapts gracefully from a 4K monitor down to a mobile device without layout shifts.
